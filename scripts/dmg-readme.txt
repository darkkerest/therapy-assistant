УСТАНОВКА Therapy Assistant
═══════════════════════════════

1. Перетащи «Therapy Assistant» в папку Applications →

2. Открой Applications → правой кнопкой по Therapy Assistant →
   нажми «Открыть» → в диалоге ещё раз «Открыть»

   (Это нужно только при первом запуске, потому что приложение
    не подписано Apple Developer ID. macOS один раз спросит
    разрешение, дальше всё будет работать как обычно.)

3. При первом запуске установщик автоматически предложит:
   • Установить BlackHole (для захвата звука из Zoom)
   • Настроить маршрутизацию аудио

   Просто следуй подсказкам на экране.

═══════════════════════════════
Что нужно для работы:
  • API-ключ Anthropic (console.anthropic.com)
  • Deepgram опционален: по умолчанию транскрипция локальная через Parakeet v3

Всё остальное приложение делает само.
